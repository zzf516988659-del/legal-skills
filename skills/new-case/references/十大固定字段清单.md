# 十大固定字段清单

## 完整字段列表

| # | 字段名 | 说明 | 填写规范 |
|---|---|---|---|
| 1 | 内部案号 | 我方内部归档编号 | 格式：2026-XXX（年份-序号），自动顺序生成 |
| 2 | 官方案号 | 法院出具案号 | 严格按法院通知书填写，含（2026）苏0685民初XXX号 |
| 3 | 案件名称 | 我方 vs 对方 + 案由 | 格式：[原告] vs [被告] [案由]，如：沈美娟 vs 海安尚鸿酒店侵权纠纷 |
| 4 | 案由 | 纠纷类型 | 从以下选取：侵权纠纷 / 合同纠纷 / 劳动争议 / 建设工程纠纷 / 股权纠纷 / 追索劳动报酬 / 经济补偿金 / 工伤保险待遇 / 民间借贷 / 买卖合同 / 服务合同 / 租赁合同 / 委托合同 / 其他 |
| 5 | 案件类型 | 一级分类 | 民事 / 劳动 / 执行 / 刑事附带民事 / 行政 |
| 6 | 立案法院 | 法院+审判庭 | 填写示例：海安市人民法院开发区法庭 / 南通市中级人民法院 |
| 7 | 立案日期 | 收到传票/材料日期 | 格式：YYYY-MM-DD |
| 8 | 案件来源 | 来源渠道 | 子公司上报 / 法务自收 / 外部委托 / 律师转介 |
| 9 | 所属分公司 | 我方主体所属公司 | 润南置业 / 尚鸿酒店 / 信拓物业 / 集团总部 / 信拓建设 / 其他 |
| 10 | 我方代理律师 | 律师姓名+联系方式 | 格式：XXX（电话），如有两名律师须注明特别授权/一般授权 |

## 案件状态十阶段流转

```
待建档 → 待答辩 → 已答辩 → 待开庭 → 已开庭 → 待判决 → 已判决 → 上诉中 → 执行中 → 已归档
```

## 期限节点计算规则（02_时间轴.md 用）

| 节点 | 计算方式 | 倒计时说明 |
|---|---|---|
| 收案日期 | 立案日期（法院送达日） | 基准日 |
| 答辩期限截止日 | 立案日期 + 15天 | 还剩X天 |
| 举证期限截止日 | 立案日期 + 30天（或证据通知收到后+7天） | 还剩X天 |
| 证据交换日 | 如法院通知，填具体日期 | 还剩X天 |
| 开庭日期 | 以传票日期为准 | 还剩X天（醒目标注） |
| 宣判预估日 | 开庭日后约30天 | 预估 |
| 判决送达日 | 以收到判决书日期为准 | — |
| 上诉期限截止日 | 判决送达日 + 15天 | 还剩X天 |
| 履行期限截止日 | 判决生效日 + 判决书规定期限 | 还剩X天 |

## 分公司→诉讼地位速查

| 分公司 | 典型诉讼地位 | 说明 |
|---|---|---|
| 润南置业 | 被告为主 | 物业租赁合同纠纷常见 |

| 集团总部 | 视情况 | 股权/投融资纠纷为主 |
