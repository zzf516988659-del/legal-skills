## 文件命名规范

### 管理文件（00 日程管理）

格式：`[案号（无括号）] [文件名].md`
示例：`2026苏0685民初1240号 案件信息.md`, `2026苏0685民初1240号 时间轴.md`

### 法律文书（06 法律文书）

格式：`[文书类型]_[案件简称]_[版本]_[日期].docx`
示例：`答辩状_沈美娟案_被告视角_20260513.docx`, `代理词_完整版_20260513_v2.docx`

### 原始材料/证据（04/05）

格式：`[材料类型]_[日期]`或`[材料来源]_[日期].格式`
示例：`鉴定报告_20260501.pdf`, `传票_20260508.pdf`

## 案件名称命名规则

```
格式：[原告简称] vs [被告简称] [案由]
规则：
- 去掉所有标点符号
- 原告/被告简称取对方当事人全称最后一个关键词
- 总长度不超过20个汉字
- 案由取核心词

示例：
- 沈美娟 vs 海安尚鸿酒店 侵权纠纷
- 左亚青 vs 润南置业 劳动争议
- 曾庆明 vs 杨小玉 信拓集团 合同纠纷
```

## ⚠️ 禁止事项

- **禁止**在案件根目录直接存放文件（必须归档到对应编号目录）
- **禁止**使用旧版目录名（03_法律文书/05_应诉方案/原始材料/07_备注等）
- **禁止**创建 README 或额外说明文档（案件信息.md 已包含完整档案说明）
