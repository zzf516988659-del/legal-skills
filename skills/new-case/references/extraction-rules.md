# 案件信息提取规则

从不同类型的材料中提取关键信息，填入案件信息看板或项目信息卡片。

## 通用原则

- **提取不虚构**：只填材料中明确出现的信息，不推测
- **标注待确认**：信息不确定时标注 `[待确认]`，不猜测
- **多来源合并**：同一信息在多个材料中出现时，以最权威来源为准（法院文书 > 合同 > 聊天记录）
- **日期格式统一**：统一使用 `YYYY-MM-DD` 格式

## 诉讼案件信息提取

### 从传票/应诉通知书提取

| 提取字段 | 查找方式 |
|----------|----------|
| 法院名称 | 抬头 / 落款印章 |
| 法院案号 | 格式如 `(2026)京0105民初1234号` |
| 开庭时间 | "定于XXXX年XX月XX日" |
| 法院地址 | 传票正文 |
| 答辩期限 | 传票发出日 + 15日 |
| 举证期限 | 传票注明 或 法院立案日 + 30日 |

### 从起诉状提取

| 提取字段 | 查找方式 |
|----------|----------|
| 原告信息 | "原告"段：姓名/公司、身份证号/信用代码、地址、电话 |
| 被告信息 | "被告"段：同上 |
| 案由 | 标题或"案由"行 |
| 诉讼请求 | "诉讼请求"段：逐条列出，注意金额 |
| 涉案金额 | 诉讼请求中的金额合计 |
| 事实与理由 | "事实与理由"段：提取关键事实和时间节点 |

### 从合同提取

| 提取字段 | 查找方式 |
|----------|----------|
| 当事人 | 甲方/乙方信息 |
| 合同标的 | 合同标的条款 |
| 合同金额 | 金额/价款条款 |
| 签订日期 | 合同首部或尾部日期 |
| 违约条款 | 违约责任章节 |
| 争议解决 | 管辖条款（约定法院/仲裁） |

### 从委托材料提取

| 提取字段 | 查找方式 |
|----------|----------|
| 委托人信息 | 委托合同首部 |
| 承办律师 | 授权委托书 / 委托合同 |
| 律师费 | 委托合同收费条款 |
| 委托范围 | 授权委托书授权范围 |

## 咨询项目信息提取

### 从咨询录音/转写提取

| 提取字段 | 查找方式 |
|----------|----------|
| 客户姓名 | 自我介绍部分 |
| 联系方式 | 录音中提到的电话/微信 |
| 对方当事人 | 提到的对方姓名/公司 |
| 咨询事项 | 核心诉求概括（1-3个关键词） |
| 关键事实 | 按时间线整理关键事件 |
| 初步判断 | 律师在沟通中给出的意见 |
| 涉及金额 | 提到的金额/损失 |
| 证据情况 | 客户提到的已有证据 |

### 从微信聊天记录提取

| 提取字段 | 查找方式 |
|----------|----------|
| 沟通时间 | 消息时间戳 |
| 沟通内容摘要 | 按主题分段概括 |
| 关键承诺/陈述 | 对方或客户的关键表述 |

## 信息不完整时的处理

| 场景 | 处理方式 |
|------|----------|
| 当事人姓名已知但身份证号未知 | 填姓名，身份证号标注 `[待确认]` |
| 法院案号未收到 | 留空，不填 |
| 涉案金额有争议 | 填最大值并标注 `[争议中]` |
| 当事人有多个角色 | 按"我方代理"和"对方"二分归类 |
| 材料日期模糊（如"去年"） | 根据材料上下文推算，标注 `[推算]` |
| 信息前后矛盾 | 以最新材料为准，标注 `[以XX材料为准]` |
