# 材料分类详细指南

本文档补充 `assets/*.yaml` 中 `classification` 关键词匹配之外的分类决策逻辑。

## 分类原则

1. **顺序匹配**：按 YAML 中 `classification` 列表顺序，命中第一条即停止
2. **文件名优先**：匹配对象为文件名（含扩展名），不区分大小写
3. **内容辅助**：文件名无法判断时，可读取文件内容前几行辅助判断

## 模糊场景处理

### 合同属于委托材料还是证据？

以 YAML 第一条匹配为准。`委托合同`、`委托协议` 匹配到委托材料（01），普通 `合同` 匹配到证据材料（05/02）。

### 起诉状在咨询项目中的归类

咨询项目中，起诉状是客户提供的材料，归入 **02-证据材料**。
诉讼项目中，律师起草的归 **06-法律文书**，对方提交的归 **07-对方提交**，法院送达的归 **08-法院送达**。

### 视频文件归类

现场录像（.mp4）归入 01-咨询记录，与客户沟通场景相关。如视频内容为证据固定（如拍摄实物证据），则归入 02-证据材料。

### 法律问答提取文件

从咨询沟通中提取的法律问答（文件名含`法律问答`或`问答提取`）归入 03-工作文件，属于律师分析产出。

### 内部沟通记录

文件名含`内部沟通`或`【内部沟通】`的文件归入 03-工作文件。常见场景：请其他律师过目方案、内部讨论案件策略。

### 转化后的子案件文件夹

咨询项目转化为诉讼案件后，可能产生以法院案号命名的子文件夹（如`（2026）苏0591民初5426号 劳动合同纠纷/`）。处理方式：
- 保留在咨询项目根目录下，不归类到 01/02/03
- 在项目信息文件的"关联案件"部分记录子案件编号和路径
- 子文件夹内部结构遵循 litigation 预设

### 录音与转写文稿配对

转写文稿（.md）与对应录音（.mp3/.aac）必须放在**同一目录**。如果录音归类到 01-咨询记录，转写也放 01。

### 无法判断的文件

归入最相关的目录，在案件信息文件中标注。如完全无法判断：
- 诉讼案件 → `{04 - 📤 客户提供}`
- 咨询项目 → `{02 - 📎 证据材料}`

## 诉讼 vs 咨询分类差异

| 场景 | 诉讼归类 | 咨询归类 |
|------|----------|----------|
| 起诉状 | 06-法律文书 或 08-法院送达 | 02-证据材料 |
| 委托合同 | 01-委托材料 | 02-证据材料 |
| 法律服务方案 | 01-委托材料 或根目录 | 根目录 或 03-工作文件 |
| 咨询录音 | 一般不出现 | 01-咨询记录 |
