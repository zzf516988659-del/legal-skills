# ARCHITECTURE.md 生成指南

Claude 应基于项目目录结构和技术栈分析生成。

**范例参考：** 读取 `references/example-funes-AGENTS.md` 中的架构分层与命令边界部分，理解如何描述系统分层。

**应从项目分析中提取：**
- 依赖文件 → 技术选型表（层级 / 技术 / 说明）
- src/ 目录扫描 → 核心模块列表及职责描述
- 入口文件分析 → 数据流描述（Mermaid 或 ASCII）
- 前后端分离的项目 → 分别描述各层的职责和边界

**质量目标：** 让新协作者读完后能理解"代码在哪里、各部分做什么、数据怎么流动"。
