# DESIGN.md 生成指南

仅前端项目生成。采用九段式结构（参考 Funes 的 DESIGN.md）：

1. Visual Theme & Atmosphere
2. Color Palette & Roles（CSS Token 表）
3. Typography
4. Layout Architecture（ASCII 布局图）
5. Component Specs
6. Interaction Rules
7. Content Rules
8. Depth & Elevation
9. Responsive Behavior

**应从项目分析中提取：**
- 依赖中的 UI 库 / CSS 框架 → 组件库和样式方案
- tailwind.config → 色彩系统
- 目录结构 → 布局架构
