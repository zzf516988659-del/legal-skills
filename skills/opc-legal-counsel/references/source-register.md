# 法源与政策来源登记表

本文件用于登记 `opc-legal-counsel` 中会被反复引用的法律、政策、标准和地方材料来源。运行时不需要整份加载；只有在回答涉及法律依据、时间敏感规则、地方覆盖层或官方核验时才读取。

## 使用规则

- 先区分来源层级：全国法律 / 部门规章 / 国家标准 / 官方公告 / 地方政策 / 归档材料
- 先引用全国通用规则，再叠加地方覆盖层
- 对备案名单、登记名单、税率、平台规则、地方窗口口径等易变事项，不凭记忆给最终结论
- 回答中引用时间敏感来源时，写明“截至核验日期”或标注“需核验”
- `archive/` 中的 OCR 原文只用于核对原始表述，不作为默认依据
